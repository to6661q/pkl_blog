## 📖 About This Project

Hi! 👋 Welcome to my **PKL Blog** repository.

I built this web-based platform during my **Praktik Kerja Lapangan (PKL)** — or Field Work Practice — internship at **Bojonegara District, Serang Regency, Banten**. The main goal of this project is to create a centralized information portal that can manage articles, publish news updates, and document daily internship activities in a structured and accessible way.

This blog system is designed to be lightweight yet functional, with a focus on clean content management and a smooth user experience. It reflects what I've learned throughout the internship — from backend logic with Laravel to frontend presentation with Blade and a touch of Vue.js.

> 💡 **Fun fact:** This project started as a simple idea to digitize the internship reporting process, and it grew into a full-fledged blog platform with multi-user support, categorization, and an admin dashboard.
